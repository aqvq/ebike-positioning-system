# README

### 项目简介

created by Shang Wentao

本项目是电动自行车定位监测系统的硬件设计部分。

满足但不限于以下需求：

- 在线调试和程序烧录接口
- 上位机控制与调试接口
- TYPEC供电接口，同时预留备用电源接口
- 丰富的调试测试点、零电阻测试点
- 极大程度实现低功耗：C800M电源开关可控、有源天线电源开关可控、USB电源开关可控
- 在重要位置添加ESD二极管，增强设备安全与稳定性

本项目使用立创EDA制作。
